## ✍️ 문제 풀이 방식

각 문제 폴더에는 다음과 같은 파일들이 포함되어 있습니다:

- `문제이름.java` : 실제 문제 풀이 코드
- `README.md` : 문제 링크, 분류, 시간/메모리 성능 요약 등 정리된 정보

예시:
📁 10026. 적록색약
┣ 📄 README.md
┗ 📄 적록색약.java
